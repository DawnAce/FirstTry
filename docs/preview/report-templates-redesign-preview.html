设计说明 / 待你拍板：
① 按「类别」分组（北京邮发 / 北京报零 / 广州日报 / 成都杂志铺 / 国图贸 / 社用报 / 合订本），每组一个组头：类别标签 + 频率（每周/每月/每年）+「N 项 · 合计 X 份」小计 + 「在此类别新增」。这样一眼看清模板结构，和印数报数里的报表分组一致 —— 原来那列到处重复的「类别」蓝标签就不用了。
② 顶部 4 张统计卡（模板项目 / 类别 / 变动项 / 默认值合计）沿用印数报数卡片样式。
③ 「变动项」由原来的 是/否 改为语义更清楚的 变动 / 固定 标签（变动=每期需重填，固定=常年不变）。
④ 每行左侧加拖拽手柄 ⠿，可在类别内拖动改排序（替代手填 sort_order 数字；数字仍保留可见）。
⑤ 频率来自前端既有常量（印数报数在用），纯前端、无需改后端；本页所有改动都不动数据库。
